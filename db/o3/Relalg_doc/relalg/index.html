About RelasJonsalgebra

RelasJonsalgebra ("relational algebra" in Norwegian, with a capital J, since my name is Jon) is a small (2.5 kloc) application for constructing and executing graphical relational algebra expressions in Oracle. If you don't have an Oracle database at hand, you may use my example database—user name and password are both "relalg". I hope the program is fairly intuitive; if not, and you can read Norwegian, I have made a manual in Norwegian. If you don't know Norwegian, you can send me an e-mail (dnalggeh.noj@ntnu.no) if you have any problems.
